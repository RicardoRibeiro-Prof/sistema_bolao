COMO GERAR O APK NOVAMENTE

1. Suba estes arquivos no GitHub Pages.
2. Aguarde alguns minutos.
3. Abra o site no navegador e confira se aparece a tela de login.
4. Entre no PWABuilder.
5. Cole o link do GitHub Pages.
6. Gere o APK novamente.
7. No celular, desinstale o APK antigo.
8. Instale o APK novo.

Esta versão não salva sessão.
Ao abrir o app, ele deve pedir e-mail e senha.
